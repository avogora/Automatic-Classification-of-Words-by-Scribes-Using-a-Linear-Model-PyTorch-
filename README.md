# Automatic-Classification-of-Words-by-Scribes-Using-a-Linear-Model-PyTorch
Название: Линейная модель классификации слов по писцам на основе частотного распределения

Цель проекта: 

Разработать и обучить модель машинного обучения, которая по частотному распределению слова у разных писцов автоматически определяет, какому писцу это слово более характерно.

Задачи: 

1. Провести предварительную обработку текстовых данных (очистка, векторизация).
2. Преобразовать тексты в числовые признаки.
3. Разделить данные на обучающую и тестовую выборки.
4. Обучить модель под задачи классификации.
5. Оцененить качество модели с помощью метрики accuracy.
6. Визуализировать результаты (scatter plot и анализ ошибок).

Описание данных: 

Файл `VEC2.csv` содержит:
Строки — писцы // Столбцы — слова // Значения — частоты слов

После обработки: каждая строка представляет одно слово.
Признаки — частоты слова у каждого писца // Метка класса — писец с максимальной частотой (argmax).

Постановка задачи:

Для каждого слова формируется вектор признаков:
`x = [freq_scribe_0, freq_scribe_1]`

Целевая метка:
`y = argmax(freq)`

Используемые технологии:

- PyTorch
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

Этапы работы:

1. Загрузка данных

Данные считываются из CSV-файла с помощью pandas.

2. Транспонирование матрицы

3. Формирование меток

Метки создаются с помощью:
`y = np.argmax(X, axis=1)`

4. Разделение данных

80% — обучение // 20% — тестирование

5. Нормализация признаков

Применяется `StandardScaler` для стандартизации частот.

6. Обучение модели

Используется линейная модель: `nn.Linear(2, 2)`
Функция потерь: `CrossEntropyLoss`
Оптимизатор: `Adam`

Результаты:

Точность на тестовой выборке:
`Accuracy ≈ 0.96`

Визуализация:
- Scatter plot
